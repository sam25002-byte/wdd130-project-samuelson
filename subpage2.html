Lock Tumblers

Lock tumblers are small mechanisms inside a lock that create the unique “key pattern.” These components determine whether a key can turn the lock. The following examples show different types of tumblers and how they function inside common lock systems.

Mechanism

How Pin-and-Tumbler Locks Work

Pin-and-tumbler locks use spring-loaded pins that align when the correct key is inserted. This design is the most common system found in homes and businesses.
